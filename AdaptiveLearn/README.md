# AdaptiveLearn - Documentación del proyecto

Este espacio reúne la documentación viva del proyecto **AdaptiveLearn**, una plataforma de software educativo orientada a la personalización del aprendizaje mediante IA generativa y diseño instruccional basado en competencias.

## Propósito de esta documentación

Organizar, clarificar y evolucionar la idea del proyecto desde sus fundamentos pedagógicos, funcionales y técnicos, de forma que sirva para:

- definir el problema educativo real que se desea resolver;
- alinear la solución con un enfoque basado en competencias;
- convertir la idea en una propuesta de producto viable;
- facilitar futuras decisiones de diseño, desarrollo y validación.

## Recomendación de uso

El orden sugerido para trabajar es:

1. completar el documento base del proyecto;
2. precisar la lógica pedagógica y de personalización del sistema;
3. responder las preguntas críticas de negocio, pedagogía y tecnología;
4. definir el MVP transversal;
5. definir perfil, escalas y reglas adaptativas;
6. organizar actores, datos y visibilidad;
7. traducir la visión a requisitos funcionales y no funcionales;
8. escoger y documentar un caso piloto inicial.

## Criterio de actualización

Cada vez que cambie una idea importante del proyecto, actualizar primero:

- la propuesta de valor;
- los usuarios principales;
- el núcleo del producto;
- el alcance del MVP;
- el caso piloto definido;
- los supuestos y riesgos.
